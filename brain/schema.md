# Brain Schema

## Page format
```markdown
# Title
updated: YYYY-MM-DD
tags: [comma-separated]
related: [links to other pages]

[content — structured lists, tables, compressed phrasing]
```

## Guidelines
- One concept per page, <400 lines
- Token-efficient: machine-first, human-readable
- Cross-link via `related:` field
- Register all pages in `brain/wiki/index.md`
