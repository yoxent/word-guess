Modify `components/game/Tile.tsx` with the following changes. Apply ALL changes in one pass:

1. **Import `useSettingsStore`** at the top:
   ```typescript
   import { useSettingsStore } from '../../stores';
   ```

2. **Inside Tile component** — DO NOT add reduceMotion or colorBlindMode as props. Instead read them directly from the store:
   ```typescript
   const colorBlindMode = useSettingsStore((s) => s.colorBlindMode);
   const reduceMotion = useSettingsStore((s) => s.reduceMotion);
   ```
   This avoids redundant prop threading (per checker issue #6). The component reads store directly, same as Confetti.tsx and ResultModal.tsx.

3. **AccessibilityLabel function** — Add inside the component or module-level:
   ```typescript
   function getAccessibilityLabel(letter: string, feedback: TileFeedback, index: number): string {
     const position = index + 1;
     if (letter === ' ' || letter === '') {
       return `Position ${position}: empty`;
     }
     const state = feedback === 'empty' ? 'active' : feedback;
     return `Position ${position}: ${letter.toUpperCase()}, ${state}`;
   }
   ```

4. **Modify animation useEffect:** At the beginning of the effect body, add:
   ```typescript
   if (reduceMotion) {
     flipProgress.value = 1;
     scale.value = 1;
     return;
   }
   ```
   This makes the effect exit immediately when reduceMotion is true, leaving shared values at their final state.

5. **Present tile text color fix (D-180):** The letter text color for present tiles must be `#1a1a2e` instead of `colors.textInverse` (white). Modify the `animatedTextStyle` or the static color on the letter element:
   - The current `styles.letter` uses `color: colors.textInverse` (white)
   - For present tiles, this must be `#1a1a2e`
   - Strategy: In the component body, compute a dynamic text color:
     ```typescript
     const letterColor = feedback === 'present' ? '#1a1a2e' : colors.textInverse;
     ```
     Then apply it on the Animated.Text via inline style override.
   - Note: `colors` here is the deprecated alias from colors.ts — it will be migrated to useColors() in Plan 06-05.

6. **Texture overlay rendering** — After the existing Animated.Text (and still inside Animated.View), render conditionally:
   ```typescript
   {colorBlindMode && feedback !== 'empty' && (
     <View style={[StyleSheet.absoluteFill, styles.textureContainer]} pointerEvents="none">
       {feedback === 'correct' && (
         <>
           <View style={[styles.dot, { top: '25%', left: '50%', marginLeft: -3, marginTop: -3 }]} />
           <View style={[styles.dot, { top: '60%', left: '25%', marginLeft: -3, marginTop: -3 }]} />
           <View style={[styles.dot, { top: '60%', left: '75%', marginLeft: -3, marginTop: -3 }]} />
         </>
       )}
       {feedback === 'present' && (
         <View style={[StyleSheet.absoluteFill, { transform: [{ rotate: '45deg' }] }]}>
           <View style={[styles.stripeBar, { top: '20%' }]} />
           <View style={[styles.stripeBar, { top: '50%' }]} />
           <View style={[styles.stripeBar, { top: '80%' }]} />
         </View>
       )}
       {feedback === 'absent' && (
         <View style={[StyleSheet.absoluteFill, { backgroundColor: 'rgba(0,0,0,0.15)' }]} />
       )}
     </View>
   )}
   ```

7. **Add new StyleSheet entries:**
   ```typescript
   textureContainer: {
     borderRadius: layout.tileBorderRadius,
     overflow: 'hidden',
   },
   dot: {
     position: 'absolute',
     width: 6,
     height: 6,
     borderRadius: 3,
     backgroundColor: 'rgba(255,255,255,0.7)',
   },
   stripeBar: {
     position: 'absolute',
     left: 0,
     right: 0,
     height: 2,
     backgroundColor: 'rgba(255,255,255,0.35)',
   },
   ```

8. **Add accessible props to Animated.View:**
   ```typescript
   <Animated.View
     accessible={true}
     accessibilityLabel={getAccessibilityLabel(letter, feedback, index)}
     accessibilityRole="text"
     ...
   >
   ```

9. **IMPORTANT:** The parent `tileStyle` must have `overflow: 'hidden'` to prevent texture overlay clipping outside `borderRadius`. Add `overflow: 'hidden'` to the `tileStyle` object.

10. **IMPORTANT:** The `colors` import from `../../constants/colors` remains unchanged — the deprecated `colors` alias in colors.ts provides backward compat. The migration to useColors() happens in Plan 06-05.

Do NOT modify GameBoard.tsx props passing — Tile reads directly from store, no props needed.
