<action>
1. In `types/settings.ts`, extend the `AppSettings` interface by adding:
   - `colorBlindMode: boolean`
   - `reduceMotion: boolean`
   - `themeMode: 'light' | 'dark' | 'system'`

2. In `stores/settingsStore.ts`, extend the `SettingsState` interface (which extends AppSettings) with 3 new actions:
   - `toggleColorBlindMode: () => void` — toggles colorBlindMode boolean
   - `toggleReduceMotion: () => void` — toggles reduceMotion boolean
   - `setThemeMode: (mode: 'light' | 'dark' | 'system') => void` — sets themeMode

3. Add default values to the `persist` initial state:
   - `colorBlindMode: false` (D-156: OFF by default)
   - `reduceMotion: false` (D-163: OFF by default)
   - `themeMode: 'system'` (D-184: 3 modes; D-185: persisted to MMKV)

4. Add the three actions to the store. Pattern matches existing toggleHardMode/toggleSound/toggleHaptic:
   - `toggleColorBlindMode: () => set((s) => ({ colorBlindMode: !s.colorBlindMode }))`
   - `toggleReduceMotion: () => set((s) => ({ reduceMotion: !s.reduceMotion }))`
   - `setThemeMode: (mode) => set({ themeMode: mode })`

Do NOT modify any other files.
</action>
