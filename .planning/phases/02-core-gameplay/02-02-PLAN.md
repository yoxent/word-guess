    **Behavior per key type:**

    Letter keys (A-Z):
    - Calls `addLetter(key)` on press (D-64)
    - Disabled when `currentGuess.length >= session.letterCount` OR `session.status !== 'playing'` OR `isRevealing` (D-66)
    - Background color from `session.keyColors[key]` (or 'unused' if not in map)
    - Color mapping: `'correct' → colors.keyCorrect (#6aaa64)`, `'present' → colors.keyPresent (#c9b458)`, `'absent' → colors.keyAbsent (#787c7e)`, `'unused' → colors.keyUnused (#d3d6da)`
    - Letter text: white (`colors.textInverse`) for colored keys, dark (`colors.keyText`) for unused keys

    ENTER key:
    - Calls `submitGuess()` on press
    - Disabled when `currentGuess.length < session.letterCount` OR `session.status !== 'playing'` OR `isRevealing`
    - Special background: `colors.keySpecial (#818384)` — wider key (use style override)
    - Text: "ENTER" in white (`colors.textInverse`), fontSize 12

    BACKSPACE key:
    - Calls `removeLetter()` on press (D-65)
    - Disabled when `currentGuess.length === 0` OR `session.status !== 'playing'` OR `isRevealing`
    - Same special background as ENTER
    - Text: "⌫" (backspace symbol), fontSize 16

    **Container styling:**
    - Full width, padding horizontal from layout, bottom padding
    - Each row: `flexDirection: 'row'`, `justifyContent: 'center'`, `gap: layout.keyboardKeyGap`
    - Row margin bottom: 4-6px
    - Keys: height = `layout.keyboardKeyHeight`, min-width = `layout.keyboardKeyMinWidth`, borderRadius = `layout.keyboardKeyBorderRadius`, paddingHorizontal for wide keys
