    **A) `src/services/wordLogic.ts` — Pure game logic functions**

    Import: `import type { GuessFeedback, TileFeedback } from '@/types';`

    Function 1: `evaluateGuess(target: string, guess: string): GuessFeedback[]`
    - Input: target word (uppercase), guess (uppercase), both same length
    - Implement Wordle duplicate-letter rules:
      1. First pass: mark exact matches as `correct` (green). Track remaining target letter counts in a `remaining: Record<string, number>` map (decrement each exact match).
      2. Second pass: for non-exact-match positions, if the letter has remaining count > 0, mark `present` (yellow) and decrement count. Otherwise mark `absent` (slate).
    - Return array of `GuessFeedback[]` same length as guess
    - Pure function — no side effects, no state
    - Return `[]` for invalid input (empty strings, mismatched lengths)

    Function 2: `validateHardMode(previousFeedback: GuessFeedback[][], newGuess: string): { valid: boolean; reason?: string }`
    - Input: all feedback from previous guesses (array of arrays) and the new guess string (uppercase)
    - NYT Wordle Hard Mode rules:
      1. All green (correct) tiles from the MOST RECENT guess must appear in the same positions in newGuess
      2. All yellow (present) tiles from ALL previous guesses must appear somewhere in newGuess
      3. For duplicate letters: if a letter was marked yellow once, it must appear at least once in the new guess. If a letter was marked green in one position and yellow in another from the same previous guess, both constraints apply independently.
    - Return `{ valid: true }` if passes, or `{ valid: false, reason: 'Must reuse confirmed tiles' }` with a specific reason string
    - Pure function

    Function 3: `isValidGuess(word: string, validWordList: string[]): boolean`
    - Case-insensitive check if word exists in the valid word list
    - Simple: create a Set from the array, test membership
    - Pure function

    **B) `src/services/dailySeed.ts` — Daily word index computation**

    Function 1: `getDailyDateString(date?: Date): string`
    - Returns UTC date string in `YYYY-MM-DD` format
    - Uses `Date.UTC()` to normalize, never device local time
    - If no date argument, uses `new Date()`

    Function 2: `getDailyWordIndex(dateStr: string, length: number, wordCount: number): number`
    - Multi-source hash approach per D-25:
      - Combine `constant + dateStr + ':' + length`
      - Compute SHA256 hash using a simple string hashing algorithm (DJB2 or built-in Web Crypto API if available in Hermes — see note below)
    - Return `Math.abs(hash) % wordCount` — deterministic index
    - Pure function

        let hash = 5381;
        for (let i = 0; i < str.length; i++) {
          hash = ((hash << 5) + hash) + str.charCodeAt(i);
          hash = hash & hash; // Convert to 32-bit integer
        }
        return Math.abs(hash);
      }

    ```typescript
    submitGuess: () => {
      const { session, currentGuess } = get();
      if (!session || session.status !== 'playing') return;

      const word = session.word;
      const guess = currentGuess.toUpperCase();
      const len = session.letterCount;

      // Validate word length (should never fire if UI is correct)
      if (guess.length !== len) return;

      // Validate guess is in dictionary (D-49) — check valid word list
      const dictStore = useDictionaryStore.getState();
      if (!dictStore.isValidGuess(len, guess)) {
        set({ error: 'Not in word list' });
        return;
      }

      // Hard Mode validation (D-59, D-60, D-61)
      if (session.hardMode && session.guesses.length > 0) {
        const hardModeCheck = validateHardMode(session.feedback, guess);
        if (!hardModeCheck.valid) {
          set({ error: hardModeCheck.reason || 'Must reuse confirmed tiles' });
          return;
        }
      }

      // Evaluate feedback
      const feedback = evaluateGuess(word, guess);

      // Build new keyColors (D-62: accumulated, correct > present > absent priority)
      const prevKeyColors = session.keyColors;
      const newKeyColors: Record<string, TileFeedback> = { ...prevKeyColors };
      const COLOR_PRIORITY: Record<string, number> = { correct: 3, present: 2, absent: 1, empty: 0 };
      for (const f of feedback) {
        const letter = f.letter;
        const existing = newKeyColors[letter];
        if (!existing || (COLOR_PRIORITY[f.feedback] || 0) > (COLOR_PRIORITY[existing] || 0)) {
          newKeyColors[letter] = f.feedback;
        }
      }

      const newGuesses = [...session.guesses, guess];
      const newFeedback = [...session.feedback, feedback];

      // Win/loss detection (GAME-04)
      const isWon = guess === word;
      const isLost = newGuesses.length >= session.maxAttempts;

      set({
        session: {
          ...session,
          guesses: newGuesses,
          feedback: newFeedback,
          keyColors: newKeyColors,
          status: isWon ? 'won' : isLost ? 'lost' : 'playing',
          completedAt: isWon || isLost ? new Date().toISOString() : undefined,
        },
        currentGuess: '',
        error: null,
      });
    }
    ```
